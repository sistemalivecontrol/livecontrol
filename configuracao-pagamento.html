💜 Configuração InfinitePay

🔐 Como funciona a integração InfinitePay:

A InfinitePay usa o Checkout API — é mais simples que Mercado Pago e Asaas!

  1. Handle é sua InfiniteTag (nome de usuário no app, sem o $)
  2. Com apenas o Handle, você já gera links de pagamento
  3. Envia uma requisição POST para api.checkout.infinitepay.io/links
  4. O cliente recebe o link e finaliza o pagamento
  5. Configure um webhook para receber confirmação automática

💡 Não precisa de API Key! Apenas seu Handle já é suficiente para gerar links.

💜 InfinitePay - Checkout API
Seu nome de usuário no App InfinitePay (sem o símbolo $). É o único dado obrigatório!
💡 Exemplo: Se no app você é "$pingodegente", digite apenas "pingodegente". Seu link será: checkout.infinitepay.io/pingodegente
Só preencha se tiver credencial de integração avançada. Para links simples, deixe em branco.
💡 O Checkout API funciona apenas com o Handle. A API Key só é necessária para integrações avançadas.
Só preencha junto com a API Key/Client ID. Para links simples, deixe em branco.
