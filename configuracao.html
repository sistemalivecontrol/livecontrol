💬 Evolution API - WhatsApp
Configure os dados da Evolution API para envio de mensagens via WhatsApp

💡 Como obter os dados da Evolution API

  • Acesse o painel da sua Evolution API
  • O URL da API e o endereco do servidor (ex: https://evolution-api-o12a.onrender.com)
  • O Nome da Instancia e o nome que voce deu a instancia do WhatsApp
  • A API Key esta nas configuracoes do manager da API
